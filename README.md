# 2023년 1학기 전공기초프로젝트 팀 B01
## 주제
주소록 관리 프로그램

## 기능
- 연락처 등록
- 연락처 검색
- 연락처 수정
- 연락처 삭제
- +@

## 사용언어
- JAVA
## 사용자 인터페이스
- CLI를 통해 사용가능
- 사용자가 CLI에서 입력한 정보를 바탕으로 연락처 등록, 검색, 수정, 삭제
